Redis安装

Posted on 2020-06-06

#### Redis安装

- 解压

```

tar -zxf redis-6.0-rc1.tar.gz

```

- redis解压包挪到/user/local

```

mv redis-6.0-rc1 /usr/local/

```

- 编译 进入文件夹/usr/local/redis-6.0-rc1

```

make

```

- 安装redis 进入文件夹src

```

make install

```

- 为了方便管理 在Redis目录下创建 bin 和 etc

```

[root@VM_0_9_centos redis-4.0.9]# mkdir bin

[root@VM_0_9_centos redis-4.0.9]# mkdir etc

```

```

mv redis.conf /usr/local/redis-4.0.9/etc/

mv mkreleasehdr.sh redis-benchmark redis-check-aof redis-check-rdb redis-cli redis-server /usr/local/redis-4.0.9/bin/

```

- 启动Redis 进入 bin目录

```

[root@VM_0_9_centos bin]# ./redis-server

```

- 关闭redis

```

./redis-cli shutdown

```

- 配置后台启动

> 首先编辑conf文件，将daemonize属性改为yes（表明需要在后台运行）

再次启动redis服务，并指定启动服务配置文件

```

./redis-server /usr/local/redis-4.0.9/etc/redis.conf

```

#### 遇到的问题

- 无法远程连接

> 修改redis.conf 注释掉 bind 127.0.0.1

> protected-mode 改为 no

- Redis Error: Connection failed. (Error: connect ETIMEDOUT)
